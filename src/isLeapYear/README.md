# isLeapYear

是否是闰年

## 代码演示

### 基础用法

```js
isLeapYear(2023) // Expected Return: true
isLeapYear('2023') // Expected Return: true
isLeapYear('YEAR') // Expected Return: false
```

## 参数介绍

| params | description |       type       |
| ------ | :---------: | :--------------: |
| year   |    年份     | number \| string |
