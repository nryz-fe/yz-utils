# kebabCase

转换字符串string为kebab case(短横线命名法).

## 代码演示

### 基础用法

```js
kebabCase('Foo Bar');
// Expected Return: 'foo-bar'

kebabCase('--foo-bar--');
// Expected Return: foo-bar
```

## 参数介绍

kebabCase(word)

| options |  type   |        note        | default |
| :-----: | :-----: | :----------------: | :-----: |
|   word   |  string  |     要转化的字符     |    -     |
