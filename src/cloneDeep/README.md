# padStart

对象数组深克隆

## 代码演示

### 基础用法

```js
cloneDeep({ a:1, b:{ c:2 } }) // Expected Return: { a:1, b:{ c:2 } }

cloneDeep('nil') // Expected Return: 'nil'
```

## 参数介绍

cloneDeep(obj)

| options |  type   |        note        | default |
| :-----: | :-----: | :----------------: | :-----: |
|   obj   |  any  |     克隆对象    |    -     |
