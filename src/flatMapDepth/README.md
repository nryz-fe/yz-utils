# flatMap

扁平化数组

## 代码演示

### 基础用法

```js
flatMap(8, 2, 0) // Expected Return: '08'

```

## 参数介绍

flatMap(collection, iteratee, depth = 1)

| options |  type   |        note        | default |
| :-----: | :-----: | :----------------: | :-----: |
|   input   |  string \| number  |     输入字符/数字     |    -     |
|   length   | number  |     返回字符串的长度     |    -    |
|   pad    | boolean | 前缀字符 |  -  |
