# routeToObj

路径转层级对象

## 代码演示

### 基础用法

```js
routeToObj(8, 2, 0) // Expected Return: '08'
```

## 参数介绍

sample(input, length, pad)

| options |  type   |        note        | default |
| :-----: | :-----: | :----------------: | :-----: |
|   input   |  string \| number  |     输入字符/数字     |    -     |
|   length   | number  |     返回字符串的长度     |    -    |
|   pad    | boolean | 前缀字符 |  -  |
