# meanBy

返回输入值在设限范围内的最大有效值

## 代码演示

### 基础用法

```js
clamp(-10, -5, 5);
// => -5

clamp(10, -5, 5);
// => 5
```

## 参数介绍

meanBy(collection, iteratee)

| options |  type   |        note        | default |
| :-----: | :-----: | :----------------: | :-----: |
|   input   |  string \| number  |     输入字符/数字     |    -     |
|   length   | number  |     返回字符串的长度     |    -    |
|   pad    | boolean | 前缀字符 |  -  |
