# round

对数字四舍五入，保留精确小数

## 代码演示

### 基础用法

```js
round(0.12345) // Expected Return: 0

round(-0.20135) // Expected Return: -0.2

round(59.92345, 2) // Expected Return: 59.92
```

## 参数介绍

round(num, precision)

| options |  type   |        note        | default |
| :-----: | :-----: | :----------------: | :-----: |
|   num   | number  |     要转换的值     |         |
|   precision   | number  |     保留小数位     |    0    |
